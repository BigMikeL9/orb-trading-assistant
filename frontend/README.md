# ORB Trading Assistant Frontend

Next.js App Router frontend for the ORB Trading Assistant MVP.

## Commands

Run commands from this `frontend/` directory.

```bash
npm install
npm run dev
npm run lint
npm test
npm run build
```

Development server:

```txt
http://localhost:3000
```

## Notes

- Styled Components is used for styling.
- Vitest and React Testing Library cover frontend behavior.
- The app currently uses local state and localStorage while backend persistence
  is introduced incrementally.
